# LOOP 城市回路原型

这是 LOOP 的第一版静态前端原型，重点是产品气质和核心信息架构。

## 当前覆盖

- 三个首发城市：上海、成都、阿布扎比
- 官方兴趣地图：咖啡、喝酒、剧场、艺术、潮流文化店、兴趣体验、秘境探索
- 官方路线：每个城市包含咖啡路线、喝酒路线和其他兴趣路线
- 城市秘境：以线索方式揭开隐藏地点和城市故事
- 我的 LOOP：今日记录、心情、隐私状态、365 天年度格
- 多视图结构：今日、地图探索、城市秘境、我的分开呈现
- 城市抽屉：上海、成都、阿布扎比用城市卡片选择
- 中文优先：右上角保留轻量 `EN` 入口，当前原型先展示中文版
- 新视觉方向：大图旅行卡、街头蓝橙识别、Lookbook 式粗体排版、圆角移动端卡片

## 打开方式

需要使用本地服务启动，这样 API key 会留在本地 `.env`，不会写进浏览器代码：

```sh
npm start
```

然后打开 `http://localhost:5373`。

## API 配置

- OpenRouter key 已放在本地 `.env`。
- 前端首页推荐会请求 `/api/chat`，由本地服务转发到 OpenRouter；接口不可用时会自动使用本地动态推荐兜底。
- 首页灵感图片会优先请求公开媒体数据：Wikipedia PageImages、iTunes Search artwork、Open Library covers；拿不到时才回退到城市氛围图。
- `.env` 已加入 `.gitignore`，不要提交到代码仓库。
